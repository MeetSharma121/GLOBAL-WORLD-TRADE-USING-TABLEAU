# 📊 Global Trade & Logistics Dashboard – Tableau Project

## 🧠 Project Objective  
The goal of this project was to build **innovative and interactive Tableau dashboards** focused on key aspects of global trade 🌍. These dashboards explore insights on:

- 📦 Potential commodities  
- 🌐 Countries involved  
- 📅 Year-wise trade activity  
- 💰 Trade amount  
- ⚖️ Quantity exchanged  

The client aimed to **launch a new business unit** centered around **global trade and logistics**, with a focus on strategic regions like the **USA 🇺🇸, Canada 🇨🇦, and Australia 🇦🇺**.

---

## 📁 Dataset Overview  
- 📌 **Total Rows**: 59,090  
- 🧮 **Total Variables**: 10  
- 🛠️ **Issues Identified**: Missing values, inconsistent entries  

---

## 🧹 Data Cleaning  
- ✅ **Tools Used**: R programming language and Excel  
- 🧼 **Cleaning Steps**:
  - Removed or imputed missing values  
  - Standardized column names  
  - Filtered relevant data for USA, Canada, and Australia  
  - Formatted numeric and date fields for better visualization  

---

## 📈 Dashboard Features (Built in Tableau)  
- 🔍 Filterable by Country, Commodity, Year  
- 🌟 KPIs and Trend Visualizations  
- 📉 Trade Amount & Quantity Over Time  
- 🗺️ Geo Maps for Country-wise Insights  
- 📊 Commodity Analysis by Region  

---

## 🚀 Outcome  
The final dashboards provide a **clear, interactive view** into international trade flows, helping the client identify high-potential opportunities and make data-driven decisions 📌.
